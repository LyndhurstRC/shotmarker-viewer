How to view a session

  1. Tap Step 1 below to open the LyndhurstRC Drive folder
  2. Find the backup file you want to view (.tar) and download it
    On Android: press and hold the file → tap Download · On iPhone/iPad: tap the file → tap the share icon → Save to Files
  3. Tap Select LRC File below to load it from your Downloads folder
  4. Alternatively, export a backup directly from the Shotmarker app — any .tar backup file will work. Tap or Select Any File to browse to wherever it is saved
