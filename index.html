How to view a session

  1. Tap Step 1 above — the LyndhurstRC Drive folder will open in a new tab
  2. Find the backup file you want to view (.tar) and download it — don't just open it
    On Android: press and hold the file → tap Download · On iPhone/iPad: tap the file → tap the share icon → Save to Files
  3. Switch back to this tab and tap Step 2 below
  4. Navigate to your Downloads folder and select the .tar file you just downloaded
